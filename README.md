# June 2026 MENA & GCC SEO Agency Technical Audit Dataset

This repository contains the raw, unedited JSON data from the comprehensive technical and Agentic Browsing (LLM) audit of the top-ranking SEO agencies and experts in the Middle East, MENA, and GCC regions.

## About the Data

We scraped the SERPs for the 28 top-ranking SEO agencies and individuals in the region and subjected them to automated audits using Google's Lighthouse and DOM analysis tools. 

The audit measured:
*   **Technical Performance:** Core Web Vitals, Largest Contentful Paint (LCP), and total network payloads.
*   **Agentic Browsing Readiness:** Testing semantic HTML, accessibility trees, and DOM structure to see if modern LLMs (ChatGPT, Gemini, Perplexity) can crawl and ingest their data.
*   **Ranking Sources:** Tracing the true source of their SERP presence (e.g., Parasite SEO, directories like Clutch/Upwork, and AI Overviews).

## The Results
The dataset exposes a 100% failure rate across the board for basic technical thresholds and Agentic Browsing readiness among the highest-ranking entities.
